##### **LED-Based Cache Hit/Miss Indicator System**



###### **Abstract**

Cache memory plays a crucial role in improving processor performance by reducing memory access time. This project presents the design and implementation of an LED-based cache hit/miss indicator system to visually demonstrate cache performance.

###### Implementation

Platform: Ripes (RISC-V Simulator)

Language: RISC-V Assembly

###### Logic:

Loop 1 accesses an array for the first time (Simulating Misses).

Loop 2 accesses the same array again (Simulating Hits).

Red LED = Miss, Green LED = Hit.

###### Results

The program counts the hits and misses for an 8-element array:

Total Misses: 8

Total Hits: 8
